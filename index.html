Stanford PwdHash

PwdHash generates theft-resistant passwords. The PwdHash browser extension invisibly generates these passwords when it is installed in your browser. You can activate this protection by pressing F2 before you type your password, or by choosing passwords that start with @@. If you don't want to install PwdHash on your computer, you can generate the passwords right here.

  • Visit the Stanford project website.
  • Read the USENIX Security Symposium 2005 paper (PDF).
  • The source code is available for use, see individual files for license details.
  • Install browser add-on PwdHash for Firefox or PwdHash for Chrome.
  • This site and plugins are no longer under active development, they should only be used for historical reference.

SECURITY WARNING AS OF 2016:

  • PwdHash never made it okay to start with a weak master password, and provides weak resistance against current technology.
  • You should migrate to a modern and actively maintained system as soon as possible.
  • View the Cracking PwdHash 2016 presentation (PDF) or read the Cracking PwdHash 2016 paper (PDF).
